# Claude 中文使用说明

## 系统提示
请始终使用中文进行所有对话和代码注释。以下是在此项目中使用中文的指导原则：

### 语言要求
- 所有对话和回复必须使用中文
- 代码注释使用中文
- 错误信息和调试信息使用中文
- 文档和说明文字使用中文

### 开发规范
1. **代码注释**：所有新代码的注释必须使用中文
2. **变量命名**：变量名可以使用英文，但注释必须用中文解释
3. **函数命名**：函数名可以使用英文，但必须有中文注释说明用途
4. **文档输出**：所有生成的文档和说明文字必须使用中文

### 项目特定要求
- **API配置**：在处理API相关代码时，使用中文注释说明配置要求
- **用户界面**：所有用户可见的文本内容使用中文
- **错误提示**：错误信息和提示文字使用中文

### 注意事项
- 始终保持专业性和技术准确性
- 中文表达要清晰、简洁
- 技术术语可以适当保留英文，但需用中文注释说明
